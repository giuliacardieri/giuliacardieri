KEEP ME POSTED

A social network for postcard collectors

Uma rede social para colecionadores de cartões postais

Keep Me Posted is a social network for postcard collectors with a recommender system. The idea came from a great project called Postcrossing, this project allows strangers to exchange postcards (the real ones, made by paper and send by regular mail). I'm a Postcrosser (the name given to Postcrossing users) since 2006 and ever since I started my computer science studies I wanted to make a software that could help me and other Postcrossers to organize and show their collections.

This application goal is to help all postcard collectors to organize their collection, find other collectors to exchange postcards with, and find postcards they might like using the search engine or by checking out the postcards recommended on by the recommender system. This project is not online yet. It's possible to check out the showcase version, which is always logged in on my profile so visitors can check all the features.

Keep Me Posted é uma rede social para colecionadores de cartões postais com um sistema de recomendação. Essa ideia veio de um ótimo projeto chamado Postcrossing, esse projeto permite que pessoas desconhecidas pelo mundo troquem cartões postais (os reais, feitos de papel e enviados pelo correio). Eu sou uma Postcrosser (o nome dado a usuários do Postcrossing) desde 2006 e logo ao iniciar minha graduação em ciência da computação tive vontade de criar um software para auxiliar usuários do Postcrossing (e consequentemente eu) a organizar e mostrar digitalmente suas coleções.

O objetivo dessa aplicação é ajudar todos os colecionadores de cartões postais a organizar sua coleção, encontrar outros colecionadores para trocar postais e encontrar postais de seus interesses usando o sistema de busca ou através das recomendações do sistema de recomendação. Esse projeto não está funcionando ainda. É possível acessar uma versão de exibição, sempre logada em meu perfil, que exibe todos os dados do sistema mas não permite o cadastro ou login de usuários. Dessa forma visitantes podem acessar quase todos os recursos do sistema.

ILLUSTRAÇÕES E ANIMAÇÕES COM CSS/SCSS

A collection of CSS/Scss illustrations and animations

Uma coleção de animações e ilustrações feitas com CSS/SCSS

On June 2016 I started to search more about illustrations made with pure CSS. I thought the idea of creating something that looked good using only code was very interesting, not only for fun but also to learn more about CSS and its attributes. After a few tries I realized I could also animate those illustrations and improve a lot my code using Sass and its structures such as mixins, if, for and variables.

Codepen is my favorite place to develop those illustrations since it's an easy way to visualize your code and result at the same time, share with your friends and colleagues. This nice website picks some "pens" to share on it's main page, other social medias and newsletter, and some of my pens were picked by Codepen staff (I admit I'm always very happy when that happens!).

Em Junho de 2016 eu comecei a pesquisar mais sobre ilustrações feitas utilizando somente CSS. Achei a ideia de criar alguma coisa que é bonita usando somente código muito interessante, não somente por diversão mas também para aprender mais sobre css e seus atributos. Após algumas tentativas percebi que poderia não só criar ilustrações, como também animá-las, e melhorar meu código utilizando Sass e as estruturas suportadas por esse pré-processador, como mixins, if, for e variáveis.

Codepen é meu lugar favorito para desenvolver essas ilustrações pois possui uma forma fácil e prática de visualizar o código e seu resultado ao mesmo tempo, além da facilidade de compartilhar com seus amigos e colegas. Esse site escolhe algum de seus "pens" para compartilhar em sua página principal, mídias sociais e newsletter, e alguns de meus pens foram selecionados por funcionários do Codepen (admito que sempre fico muito feliz quando isso acontece!).

DINOKIKI

A fun website about a dinosaur that says 'kiki'

Um website divertido sobre um dinosauro que fala 'kiki'

Dinokiki is an idea inspired by the website pls pet doge and Jerome Jarre's and Nicholas Megalis' Vine "Mama Dinosaur Finds Its Baby". I really wanted to learn JavaScript and I thought using a fun project would be nice! (and I was right!) My friends supported me recording kikis while I tried to design a character and program in JavaScript for the first time.

Dinokiki é uma ideia inspirada pelo website pls pet doge e o Vine do Jerome Jarre e Nicholas Megalis "Mama Dinosaur Finds Its Baby". Atualmente o site tem funcionalidades interativas, como um formulário que permite o envio de arquivos de áudio (os "kikis") e testá-los com o dinokiki ao mesmo tempo, um jogo para identificar cada "kiki" (um arquivo de áudio enviado por um usuário) e ilustrações diferentes para cada kiki.

WHY SÃO PAULO?

A touristic website about São Paulo

Um website turístico sobre São Paulo

Why São Paulo? was a project created for my Web II course at Rochester Institute of Technology. It was developed in two steps. The first was only the static pages with HTML and CSS. Why São Paulo? foi um projeto criado para a disciplina de Web II no Rochester Institute of Technology. Foi desenvolvido em dois passos. O primeiro foi a criação de páginas estáticas com HTML e CSS. O segundo passo foi incluir JavaScript (sem bibliotecas!), PHP e MySQL. O foco nesse projeto foi boa usabilidade, código limpo e bem estruturado.

Eu tive que escolher um lugar no mundo para criar um site. A game based on specialist systems

Um jogo baseado em sistemas especialistas

Sou fã do Akinator e ao descorbir que era baseado em um sistema especialista decidi tentar criar ao similar. O usuário escolhe um professor na página inicial clicando na foto e depois responde perguntas (somente sim/não) e o sistema irá acertar o professor que o usuário escolheu. Não é mágica, é inteligência artificial!

NO CÉU TEM PÃO?

A game based on specialist systems

Um jogo baseado em sistemas especialistas

No céu tem pão? is the question "Is there bread in the sky?" in portuguese. It was created based on a famous viral video in Brazil. The website purpose is just to have fun, you need to look at the clouds and answer the question according to what you see.

The animated clouds and breads were created using CSS3 animations, the score is kept using HTML5 local storage. There is 30% of chances the visitor will see bread in the sky. It has responsive design since most of the visitors access it using a mobile device.

Everytime the user clicks on the "SIM"(yes) or "NÃO"(no) buttons the scores changes. I chose HTML5 local storage to store the results because it is a simple and easy way to retain information without using a database (which would be too complex for this simple single page game).

Esse projeto foi criado baseado no famoso vídeo viral "No Céu Tem Pão?". O objetivo desse site é se divertir, é necessário olhar para as nuvens e responder a questão de acordo com o que você vê.

As nuvens animadas foram criadas usando animações de CSS3, o placar é mantido usando o local storage do HTML5. Há uma chance de 30% do visitante ver pão no céu. Esse projeto tem design responsivo pois a maioria de seus visitantes acessa o site através de um dispositivo móvel.

Toda vez que o usuário clica nos botões "SIM" ou "NÃO" o placar muda. Escolhi utilizar o local storage do HTML5 para armazenar os resultados porque é uma forma simples e fácil de manter informações sem utilizar um banco de dados (o qual seria muito complexo para criar esse jogo de uma única página).

DEVO ESQUENTAR MEU SANDUÍCHE?

A real time weather website for Bauru

Um site com o clima em tempo real para Bauru

Devo esquentar meu sanduíche? is the question "Should I heat up my sandwich?" in portuguese. I created this website because the city of Bauru is really warm and also because there's a famous sandwich in Brazil with this city name. Its purpose is to check the current weather (temperature, rain/sun, day/night) and also to have fun looking at the animations or clicking on the clouds.

All scenery and animations were created using CSS only. There are no images. The scenery changes according to the time of the day and also if it's raining.

The weather information is obtained using the JavaScript library simpleWeather.js.

Criei esse site porque a cidade de Bauru é muito quente e também pela cidade compartilhar seu nome com o famoso sanduíche Bauru. O objetivo desse site é checar o clima atual (temperatura, sol/chuva, dia/noite) e também para ser divertido ao olhar as animações ou clicando nas nuvens.

Todo cenário e animações foram criados usando somente CSS. Imagens não foram utilizadas. O cenário muda de acordo com o horário do dia e também se estiver chovendo.

A informação sobre o clima é obtida usando a biblioteca JavaScript chamada simpleWeather.js.

TALK WITH DINOKIKI

An interactive website using speech recognition

Um site interativo usando reconhecimento de voz

Dinokiki is a project I have since February 2014. I always wanted to make a page where the user could say something and Dinokiki could "answer". On this pen I used HTML5 Speech Recognition so the user can say the name of a person or a word, from a previously defined list available on the help link, and Dinokiki will answer according to that.

'Kikis' are audios recorded by people and sent to the website. Each kiki has a name such as "The Singer Kiki" or "The Caveman Kiki". The language is set to portuguese, since most of the names are brazilian. IMPORTANT: it's supported only on desktop Google Chrome.

Dinokiki é um projeto que eu desenvolvo desde Fevereiro de 2014. Eu sempre tive vontade de criar uma página que o usuário poderia falar alguma coisa e o Dinokiki poderia "responder". Nesse pen eu usei o reconhecimento de voz do HTML5, dessa forma o usuário pode falar o nome de uma pessoa ou uma palavra, de uma lista pré-definida disponível no link "help", e o Dinokiki irá responder de acordo com o que foi falado.

"Kikis" são os áudios gravados por pessoas e enviados ao site. Cada kiki tem um nome como "The Singer Kiki" ou "The Caveman Kiki". O idioma está definido como português, pois a maioria dos nomes é brasileiro. IMPORTANTE: só é suportado pelo Google Chrome de desktop.

DINOAPP PWA

A Dinokiki PWA

Um PWA do Dinokiki

In April 2017 I was starting my graduate studies, my advisor suggested we started exploring Progressive Web App (PWA) fiels, so I created Dinoapp. It's a simple PWA developed with the purpose of understanding how a PWA works, including permissions, push notifications, service workers and security. Dinokiki is a project I have since February 2014 and I thought would be a nice to have a PWA from it.

On February and March 2018 I plan to expand this app adding better UIs and creating more features.

Em Abril de 2017 comecei meu mestrado, minha orientadora sugeriu iniciar uma exploração na área de Progressive Web App (PWA), então criei o Dinoapp. É um PWA simples, desenvolvido com o propósito de entender como um PWA funciona, incluindo o uso de permissões, notificações push, service workers e segurança. Dinokiki é um projeto que eu desenvolvo desde Fevereiro de 2014 e pensei que seria uma boa ideia ter um PWA dele.

Em Fevereiro e Março de 2018 pretendo expandir esse app, adicionando uma UI melhor e criando mais funcionalidades.
