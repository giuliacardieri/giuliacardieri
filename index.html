MINHA AGENDA

An event scheduling PWA

Um PWA para agendar eventos

Minha Agenda is part of my graduate thesis. This PWA has concepts of End-User Development (EUD), where users that are not professional developers can create, edit and extend web applications. Meta-Design is an EUD approach and it is used to allow users to become co-designers on the app. On Minha Agenda, users can customize interface elements, modalities and content display.

This PWA was developed as a prove of concept of PWA-EU, the approach for PWA applications I propose on my graduate studies. It was developed with Vue.js PWA template and Materialize. All data is stored on LocalStorage due to privacy concerns.

Minha Agenda é parte da minha dissertação de mestrado. Esse PWA possui conceitos de End-User Development (EUD), onde usuários que não são desenvolvedores profissionais podem criar, editar e estender aplicações. Meta-Design é uma abordagem de EUD e permite que usuários tornem-se co-designers da aplicação. No Minha Agenda, usuários podem customizar elementos de interface, modalidades e formato de exibição de conteúdo.

Esse PWA foi desenvolvido como prova de conceito da abordagem PWA-EU, que eu proponho em minha pesquisa de mestrado. Utilizei Vue.JS com o template PWA e Materialize. Todo conteúdo foi armazenado no LocalStorage devido a questões de privacidade.

CARDIERI INC

CARDIERI INC

A constructing company website

Um website para uma construtora

Cardieri Construtora e Incorporadora is a constructing company in Sorocaba-SP. I developed and designed a website that allows the company to show their past and current buildings, tell their history and contact information.

First, I prototyped the interfaces with Sketch in order to understand what the company needed, after that I started developing with Vue.js + Sass (using BEM methodology to organize my CSS). A REST API was created with Express/Node.js and consumed by the Vue.js project.

Cardieri Construtora e Incorporadora é uma empresa de construção em Sorocaba-SP. Eu criei e desenvolvi um site que permite que a empresa mostre seus empreendimentos atuais e passados, conte sua história e exiba suas informações de contato.

Inicialmente eu criei prototipos usando Sketch para entender as necessidades da empresa, e depois iniciei a fase de desenvolvimento com Vue.js + Sass (usando a metodologia BEM para organizar meu CSS). Uma API REST foi criada com Express/Node.js e consumida pelo projeto Vue.

TALK WITH DINOKIKI

An interactive website using speech recognition

Um site interativo usando reconhecimento de voz

Dinokiki is a project I have since February 2014. I always wanted to make a page where the user could say something and Dinokiki could "answer". On this pen I used HTML5 Speech Recognition so the user can say the name of a person or a word, from a previously defined list available on the help link, and Dinokiki will answer according to that.

'Kikis' are audios recorded by people and sent to the website. Each kiki has a name such as "The Singer Kiki" or "The Caveman Kiki". The language is set to portuguese, since most of the names are brazilian. IMPORTANT: it's supported only on desktop Google Chrome.

Dinokiki é um projeto que eu desenvolvo desde Fevereiro de 2014. Eu sempre tive vontade de criar uma página que o usuário poderia falar alguma coisa e o Dinokiki poderia "responder". Nesse pen eu usei o reconhecimento de voz do HTML5, dessa forma o usuário pode falar o nome de uma pessoa ou uma palavra, de uma lista pré-definida disponível no link "help", e o Dinokiki irá responder de acordo com o que foi falado.

"Kikis" são os áudios gravados por pessoas e enviados ao site. Cada kiki tem um nome como "The Singer Kiki" ou "The Caveman Kiki". O idioma está definido como português, pois a maioria dos nomes é brasileiro. IMPORTANTE: só é suportado pelo Google Chrome de desktop.

KEEP ME POSTED

A social network for postcard collectors

Uma rede social para colecionadores de cartões postais

Keep Me Posted is a social network for postcard collectors with a recommender system. The idea came from a great project called Postcrossing, this project allows strangers to exchange postcards (the real ones, made by paper and send by regular mail). I'm a Postcrosser (the name given to Postcrossing users) since 2006 and ever since I started my computer science studies I wanted to make a software that could help me and other Postcrossers to organize and show their collections.

This application goal is to help all postcard collectors to organize their collection, find other collectors to exchange postcards with, and find postcards they might like using the search engine or by checking out the postcards recommended on by the recommender system. This project is not online yet. It's possible to check out the showcase version, which is always logged in on my profile so visitors can check all the features.

Keep Me Posted é uma rede social para colecionadores de cartões postais com um sistema de recomendação. Essa ideia veio de um ótimo projeto chamado Postcrossing, esse projeto permite que pessoas desconhecidas pelo mundo troquem cartões postais (os reais, feitos de papel e enviados pelo correio). Eu sou uma Postcrosser (o nome dado a usuários do Postcrossing) desde 2006 e logo ao iniciar minha graduação em ciência da computação tive vontade de criar um software para auxiliar usuários do Postcrossing (e consequentemente eu) a organizar e mostrar digitalmente suas coleções.

O objetivo dessa aplicação é ajudar todos os colecionadores de cartões postais a organizar sua coleção, encontrar outros colecionadores para trocar postais e encontrar postais de seus interesses usando o sistema de busca ou através das recomendações do sistema de recomendação. Esse projeto não está funcionando ainda. É possível acessar uma versão de exibição, sempre logada em meu perfil, que exibe todos os dados do sistema mas não permite o cadastro ou login de usuários. Dessa forma visitantes podem acessar quase todos os recursos do sistema.

NO CÉU TEM PÃO?

A game based on specialist systems

Um jogo baseado em sistemas especialistas

No céu tem pão? is the question "Is there bread in the sky?" in portuguese. It was created based on a famous viral video in Brazil. The website purpose is just to have fun, you need to look at the clouds and answer the question according to what you see.

The animated clouds and breads were created using CSS3 animations, the score is kept using HTML5 local storage. There is 30% of chances the visitor will see bread in the sky. It has responsive design since most of the visitors access it using a mobile device.

Everytime the user clicks on the "SIM"(yes) or "NÃO"(no) buttons the scores changes. I chose HTML5 local storage to store the results because it is a simple and easy way to retain information without using a database (which would be too complex for this simple single page game).

Esse projeto foi criado baseado no famoso vídeo viral "No Céu Tem Pão?". O objetivo desse site é se divertir, é necessário olhar para as nuvens e responder a questão de acordo com o que você vê.

As nuvens animadas foram criadas usando animações de CSS3, o placar é mantido usando o local storage do HTML5. Há uma chance de 30% do visitante ver pão no céu. Esse projeto tem design responsivo pois a maioria de seus visitantes acessa o site através de um dispositivo móvel.

Toda vez que o usuário clica nos botões "SIM" ou "NÃO" o placar muda. Escolhi utilizar o local storage do HTML5 para armazenar os resultados porque é uma forma simples e fácil de manter informações sem utilizar um banco de dados (o qual seria muito complexo para criar esse jogo de uma única página).

BAURU

A interactive website made with css

Um site interativo feito com css

This website is based on a city called Bauru, in Brazil, it shows an illustration from Vitória Regia Park It works as a weather simulator. By changing the selected sliders the user is able to change from day to night and also from sun ro rain.

All scenery and animations were created using css only. There are no images. It's nice to simulate the real world using only css :D

Esse site é baseado na cidade de Bauru. Ele mostra uma ilustração do Parque Vitória Regia. Esse projeto funciona como um simulador do clima. Ao mudar os sliders o usuário consegue alterar o cenário de dia para noite e também de sol para chuva.

Todo cenário e animações foram criados usando somente css. Imagens não foram utilizadas. É legal simular coisas do mundo real usando somente css :D
